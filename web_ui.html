Справка

Новое в v10

  • Prometheus Метрики - мониторинг всех параметров в реальном времени по 8 категориям
  • Session Metadata - парсинг CSV/JSON сессии (HFR/FWHM статистика по фильтрам)
  • Продвинутый парсинг Sequence - читаемые имена, Expression, Triggers, Conditions, активный элемент
  • Аналитика сессии - единая вкладка с Prometheus/Session/Профилем
  • Эндпоинт /api/metrics для внешних систем мониторинга

Режимы работы AI

  • Авто - умная маршрутизация (вопросы/команды)
  • Команды - только выполнение
  • Консультант - ответы и советы
