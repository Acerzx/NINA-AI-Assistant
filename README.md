# NINA-AI-Assistant
Dashboard и AI ассистент для мониторинга и управления астрофотографией в NINA


Системный промт в Qwen:

Проект dashboard и AI ассистента, для взаимодействия с программой для астрофотографии NINA ( https://github.com/isbeorn/nina ) 

Идеи проекта: 
1) dashboard для мониторинга параметров оборудования и ключевых метрик сессии
2) реализация команд для быстрого управления оборудованием
3) чат с AI ассистентом:
	3.1) для анализа получаемых данных и поведения системы
	3.2) для консультирования по всевозможным темам сопряженных с астрофотографией
	3.3) для исполнения команд, а также цепочек команд
4) необходима полная поддержка интеграции с плагинами, для сбора всех метрик, статистик итп:
	4.1) https://github.com/prometheus/prometheus и https://github.com/jewzaam/nina-prometheus-exporter
	4.2) https://github.com/tcpalmer/nina.plugin.sessionmetadata
   
Требуется строгое и полное соответствие документации:
1) https://github.com/isbeorn/nina
2) https://github.com/christian-photo/ninaAPI
3) https://github.com/PaDev1/Nina_advanced_api_mcp
